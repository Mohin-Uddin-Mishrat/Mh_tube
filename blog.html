What do you mean by REST API?

A rest Api is Representational State Transfer Application Programming Interface , rest api is a set of rules and conventions for building and interacting with web services. It is an architectural style that follows a set of principles and constraints designed to make web services scalable, stateless, and easily maintainable

Discuss the scope of var, let, and const

var:: variables deaclared with var are function scopped, that means the variable can be visible throwout the entire function where it is declared. And they are hoisted to the top of their scop , that means the variables can be used before their declaration.

Let:: variables deaclared with Let are Block scopped, that means the variable can be visible throwout the entire block where it is declared. And they are not hoisted to the top of their scop , that means the variables can not be used before their declaration.

const:: variables deaclared with var are block scopped, that means the variable can be visible throwout the entire block where
it is declared. And they are immutable.

Tell us the use cases of null and undefined

Null :: if a function expects an object as an argument and an object is not available, you might pass null to indicate the absence of an object.
You might set a variable to null to indicate that it should no longer hold a specific value.

undefined :: When a function is called with fewer arguments than declared parameters, the missing parameters are set to undefined.
Accessing a property that doesn't exist on an object returns undefined.
